# RAF Cloud – Frontend

Frontend aplikacija za upravljanje korisnicima i virtuelnim mašinama u cloud simulaciji.

## Tehnologije
- Angular 16
- TypeScript
- Angular Router
- Reactive Forms

## Funkcionalnosti
- Prijava korisnika
- Prikaz, dodavanje i izmena korisnika
- Pretraga i filtriranje mašina
- Kreiranje novih mašina
- Prikaz istorije grešaka
- Prikaz funkcionalnosti u skladu sa korisničkim dozvolama

## Napomena
Frontend koristi mock podatke u početnoj fazi razvoja.
